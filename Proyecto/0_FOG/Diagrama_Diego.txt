¿¿¿Aplicar NAT en R1????

Opción A: Configurar NAT en R1 (La más común)

Esto hará que todo el tráfico de las LANs de la derecha salga hacia la izquierda con la IP de R1 (10.2.7.3). R-DH no necesitará saber nada de las otras redes.

Ejecuta estos comandos en R1-ubuntu:

    Instala las herramientas de iptables (si no las tienes): sudo apt update && sudo apt install iptables-persistent -y

    Aplica la regla de Masquerade (asumiendo que ens18 o la interfaz que sea es la que va hacia vmbr207): sudo iptables -t nat -A POSTROUTING -o [INTERFAZ_HACIA_VMBR207] -j MASQUERADE

        Nota: Sustituye [INTERFAZ_HACIA_VMBR207] por el nombre real (ej. eth0 o ens18). Puedes ver el nombre con ip a.
