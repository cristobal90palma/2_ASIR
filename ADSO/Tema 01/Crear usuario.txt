
COMANDOS:

Para cargar los ficheros:
   ldapadd -x -D cn=admin,dc=SUAREZ1,dc=abad2 -w "usuario.12345" -f 

Para borrar una entrada
   ldapdelete -x -W -D cn=admin,dc=SUAREZ1,dc=abad2 "ou=desorganizados,dc=SUAREZ1,dc=abad2"
   
   ldapdelete -x -W -D cn=admin,dc=SUAREZ1,dc=abad2 "cn=genterara,ou=grupos,dc=SUAREZ1,dc=abad2"

Para hacer una busqueda de una entrada:
   ldapsearch -xLLL -b dc=sistemas,dc=edu uid=santi
o también
   ldapsearch -xLLL -b dc=sistemas,dc=edu cn=profesores

Para realizar modificaciones:
   ldapmodify -x -D cn=admin,dc=SUAREZ1,dc=abad2 -w "usuario.12345" -f modificacion.ldif

Para generar las claves para los usuarios
   slappasswd
