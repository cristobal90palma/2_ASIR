1) Instalación
Para no tener problemas de versiones (Windows, Linux, MacOS) Lo instalamos en un contenedor de Docker

TIENES QUE TENER ENCENDIDO Docker DESKTOP (SI NO TIENES SOLO EL ENGINE)

docker run -d -p 8080:8080 -p 50000:50000 -v jenkins_home:/var/jenkins_home jenkins/jenkins:lts-jdk17


navegador web:

http://localhost:8080

Nos indicará que debemos usar una contraseña generada dentro del contenedor.

cat /var/jenkins_home/secrets/initialAdminPassword
